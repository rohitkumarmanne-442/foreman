🧑‍🏭 How Foreman works

Every AI agent session becomes a review card, ranked by risk. Work the inbox like email — approve what's safe, flag what isn't. Inbox zero means every agent change had human eyes on it.

Connect an agent

  • Claude Code / Cursor — run foreman init in your repo (hooks, richest data: commands, claims, diffs).
  • Any other IDE or agent — run foreman watch in the repo. Universal mode diffs against git and works with everything: Windsurf, Copilot, JetBrains AI…
  • MCP servers — prefix the server command with foreman wrap --name <srv> -- for signed receipts + rug-pull detection.

Reading a card

  • Claims vs evidence — ⚠️ UNVERIFIED means the agent said "it works" but never ran anything that could prove it.
  • Findings — destructive commands, mass rewrites, secrets in code, sensitive paths. Each explains why it matters.
  • Files — click any file for the before/after diff.
